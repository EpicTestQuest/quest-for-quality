## Usage in Wizzo Beta Testing

1. Clone this repository
2. Navigate to the screenshots folder
3. Upload any screenshot to Wizzo via DM
4. Type: "Generate test cases for this screen focusing on [focus areas]"
5. Review the AI-generated test cases
